# 扫码系统 - HTML版本

这是Flutter扫码系统的HTML网页版本，提供了与原移动应用相同的核心功能。

## 主要功能

- 用户登录（只需输入姓名）
- 单个扫码（绕线、嵌线、接线、压装、车止口、浸漆）
- 连续扫码（绕线、车止口、浸漆）
- 产品查询（按工序、型号和产品编码查询）
- 产品扫码查询（通过扫描产品二维码直接查看详情）

## 技术栈

- HTML5 / CSS3 / JavaScript
- Bootstrap 5（UI框架）
- HTML5-QRCode（扫码库）
- Supabase（数据库）

## 使用方法

1. 确保设备有摄像头权限
2. 打开index.html文件
3. 输入姓名登录
4. 使用系统功能

## 数据存储

应用使用Supabase作为后端数据库，与Flutter版本使用相同的数据表。
